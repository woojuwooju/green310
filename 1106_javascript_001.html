자바스크립트란?

브라우저에서 실행되며 브라우저를 직접 조작하거나 html,css로는 할 수 없는 기능을 가능하게 하고 실시간으로 변경이 가능하다.

입력 -> 가공 -> 출력 이라는 공통으로 사용되는 기본처리흐름을 이해하고 언제 처리를 실행할지를 정하는 이벤트를 추가하는 전체구조를 갖는다.

출력하기

  • alert() - 경고창과 같은 메세지창을 나타냅니다.
  • prompt() - 경고창과 비슷하게 메세지창이 나타내고 입력창과 함께 받아올 수 있다.
  • console.log() - 브라우저에 직접 나타나지 않고 개발창에서 확인이 가능
  • document.write() - 브라우저에 나타나는 구조

변수

특정 코드에서 얻은 데이터를 저장하거나 저장된 데이터를 변환할때 사용

변수사용법

  1. 변수를 정의한다 : 식별자를 통해 변수의 이름을 작명하고 선언한다.
    • var 식별자=대상자(가장 자유롭게 변환할 수 있는 구조)
    • const 식별자=대상자(고유명사같은개념)-외부언어를 자바스크립트 이름으로 변환시키는 구조
    • let 식별자=대상자 (var와 같음 단, 중복설정을 걸러냄)

자료형 및 연산자

자료형이란? 프로그래밍 언어에서 사용되는 구조 - 숫자, 함수, 객체, 정의되지 않은 자료

연산자란? 덧셈, 뺄셈, 곱셈, 나눗셈, 나머지를 표현하는 구조

스크립트를 통한 표현

산술연산자

말 그대로 데이터의 값을 계산하는 연산자

  • 더하기 + 데이터와 데이터를 더하거나 병합한다.
  • 빼기 - 숫자와 숫자를 뺀다.
  • 곱하기 * 숫자와 숫자를 곱한다.
  • 나누기 / 숫자와 숫자를 나눈다.
  • 나머지 % 특정값을 나눈 후의 나머지(홀수, 짝수 구분)

대입연산자

=을 기준으로 오른쪽의 숫자만큼 사칙연산하여 왼쪽변수의 데이터값을 갱신

  • += var a=5; a+=5 결과값10
  • -= var b=5; b-=2 결과값 3
  • *= var c=5; c*=5 결과값 25
  • / var d=5; d/=5 결과값 1
  • % var e=5; e%=3 결과값2

증감연산자

1씩 더하거나 1씩 뺌

  • ++ ; var i=0; i++
  • -- ; var ii=0; i--
